最長共通部分列。

# 関数

```cpp
int longest_common_subsequence_length(a, b)
Sequence longest_common_subsequence(a, b)
```

`longest_common_subsequence` は最長共通部分列を 1 つ返す。

## 時間計算量

- length: $O(NM)$
- 復元: $O(NM)$
