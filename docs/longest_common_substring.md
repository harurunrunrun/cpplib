2 つの文字列の最長共通部分文字列を求める。

## 構造体

- `LongestCommonSubstringResult`
    - `first_pos`
    - `second_pos`
    - `length`

## 関数

- `longest_common_substring(a, b)`

## 計算量

- $O((n + m) \log(n + m))$
